Ejercicio 4


El ejercicio consiste en llevar a cabo el desarrollo de un metodo que se encargue de sustraer un elemento del array.
Hay dos fases:
-La suma de los valores del Array antes de eliminar un numero de este.
-La suma de los valores del Array despues de eliminar un numero de este

El array esta compuesto de los valores 1,2,3 y 4. El valor eliminado es el 1.
